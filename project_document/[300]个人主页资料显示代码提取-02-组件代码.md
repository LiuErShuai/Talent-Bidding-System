# 个人主页资料显示代码提取 - 组件代码

## 1. ProfileCard.vue - 个人资料卡片组件

这是从 `ResumeInterview.vue` 中提取的个人资料预览卡片组件，用于展示用户的完整档案信息。

### 组件功能
- 显示用户基本信息（姓名、联系方式、教育背景等）
- 显示技术技能
- 显示工作经验（时间线形式）
- 显示项目经验
- 显示教育背景
- 显示证书和语言能力

### 完整源代码

```vue
<template>
  <!-- 用户档案预览 -->
  <el-card v-if="hasResume" class="profile-card">
    <template #header>
      <div class="card-header">
        <h2>
          <i class="fas fa-user-circle"></i>

    <div class="profile-section">

          <el-tag v-if="resumeData.personal_info?.phone" type="success">
            <i class="fas fa-phone"></i> {{ resumeData.personal_info.phone }}

          <span v-else class="not-available">暂无信息</span>
        </el-descriptions-item>

            <i class="fas fa-envelope"></i> {{ resumeData.personal_info.email }}

        </el-descriptions-item>

          <el-tag v-if="resumeData.personal_info?.major" type="primary">
            <i class="fas fa-graduation-cap"></i> {{ resumeData.personal_info.major }}
          </el-tag>
          <span v-else class="not-available">暂无信息</span>
        </el-descriptions-item>

            {{ resumeData.personal_info.expected_salary }}
          </el-tag>
        </el-descriptions-item>
      </el-descriptions>
    </div>

    <el-divider />

    <!-- 技术技能 -->
    <div class="profile-section">
      <h3><i class="fas fa-code"></i> 技术技能</h3>

              size="small"

            >

            </el-tag>

      </div>

    </div>

          :key="index"
          :label="edu.degree || '学历'"
        >
          <div class="education-item">
            <strong>{{ edu.school || edu.institution }}</strong>
            <span v-if="edu.major"> - {{ edu.major }}</span>

                :key="achievement"
                size="small"
                type="success"

                {{ achievement }}
              </el-tag>
            </div>
          </div>
        </el-descriptions-item>
      </el-descriptions>
    </div>

      <h3><i class="fas fa-plus-circle"></i> 其他信息</h3>
      <el-row :gutter="16">

          <div class="extra-info-item">
            <h4>证书认证</h4>
            <el-tag
              v-for="cert in resumeData.certifications"
              :key="cert"
              size="small"
              type="warning"
            >
              <i class="fas fa-certificate"></i> {{ cert }}
            </el-tag>
          </div>
        </el-col>
        <el-col :span="12" v-if="resumeData.languages?.length">
          <div class="extra-info-item">
            <h4>语言能力</h4>
            <el-tag
              v-for="lang in resumeData.languages"
              :key="lang"
              size="small"
              type="info"
            >
              <i class="fas fa-language"></i> {{ lang }}
            </el-tag>
          </div>
        </el-col>
      </el-row>
    </div>
  </el-card>
</template>

<script setup>
import { computed } from 'vue'

// Props定义
const props = defineProps({
  resumeData: {
    type: Object,
    required: true,
    default: () => ({})
  },
  hasResume: {
    type: Boolean,
    default: false
  }
})

// 计算属性
const hasExtraInfo = computed(() => {
  return props.resumeData?.certifications?.length ||
         props.resumeData?.languages?.length
})

// 获取技能水平对应的标签类型
const getSkillLevelType = (level) => {
  const levelMap = {
    '初级': 'info',
    '中级': 'warning',
    '高级': 'success',
    '专家': 'danger',
    '资深': 'danger'
  }
  return levelMap[level] || 'info'
}

// 根据项目名称获取项目详情
const getProjectDetail = (projectName) => {
  if (!props.resumeData?.project_experience_detail?.length || !projectName) {
    return null
  }

  return props.resumeData.project_experience_detail.find(
    detail => detail.title === projectName
  )
}
</script>
```

```vue

  <ProfileCard

    :has-resume="!!userProfile"
  />
</template>

import ProfileCard from '@/components/ProfileCard.vue'

  personal_info: {
    name: '张三',

    phone: '13800138000',

  },

</script>
```
