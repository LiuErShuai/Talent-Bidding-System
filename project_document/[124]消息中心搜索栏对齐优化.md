# [124]消息中心搜索栏对齐优化

> 背景与目标：消息中心左侧搜索栏需要与下方消息列表项（`list-item`）左右边界对齐，并避免搜索框在容器内出现裁切/显示不完整的视觉问题，提升整体一致性与观感。

## 1. 需求与验收标准
- 搜索栏输入框左右边界与下方 `list-item` 的左右边界一致（视觉对齐）。
- 搜索栏完整可见（尤其是聚焦态的阴影/描边不应被容器裁切得明显突兀）。
- 不影响现有聊天列表、互动列表、右侧聊天面板与弹窗逻辑。

## 2. 方案说明
- 保持结构不变，通过样式微调实现对齐与不裁切观感：
  - 让搜索输入在 flex 容器中使用 `flex: 1`，保证宽度填满与列表项一致。
  - 将聚焦态由“大范围投影”改为“轻量描边高亮”，减少被 `overflow: hidden` 裁切的概率。
