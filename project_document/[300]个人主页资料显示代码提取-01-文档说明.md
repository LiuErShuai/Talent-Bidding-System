# 个人主页资料显示代码提取文档

## 文档说明

本文档用于提取和复用项目中个人主页资料显示相关的代码。包含完整的源代码和使用指导。

## 文档结构

本文档分为以下几个部分：

1. **文档说明**（当前文档）- 文档概述和结构说明
2. **组件代码** - Vue组件源代码
3. **API和状态管理** - API接口和状态管理代码
4. **样式代码** - CSS样式代码
5. **使用指导** - 集成和使用说明

## 提取的代码模块

### 1. 核心组件
- **ProfileCard.vue** - 个人资料卡片组件（从ResumeInterview.vue提取）
- **UserHeader.vue** - 用户头部信息组件（从AppHeader.vue提取）
- **SettingsForm.vue** - 设置表单组件（从Settings.vue提取）

### 2. API接口
- **user.js** - 用户信息管理API

### 3. 状态管理
- **useAuth.js** - 认证状态管理（包含用户信息）

### 4. 样式系统
- **CSS变量系统** - 主题色彩和设计令牌
- **组件样式** - 个人资料显示相关样式

## 技术栈

- **Vue 3** (Composition API + `<script setup>`)
- **Element Plus** - UI组件库
- **FontAwesome 6** - 图标库
- **原生CSS + CSS变量** - 样式系统

## 依赖说明

### 必需依赖
- `vue@^3.x`
- `element-plus@^2.x`
- `vue-router@^4.x`
- `@element-plus/icons-vue`

### 可选依赖
- FontAwesome 6 (CDN引入)
