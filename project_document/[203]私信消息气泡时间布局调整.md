# 背景与目标

- **背景**：当前私信聊天气泡内将“发送时间”置于文本下方，且会占用 `chat-bubble` 内部空间，不符合 QQ/微信式“时间贴边显示”的视觉。
- **目标**：将发送时间移动到消息文本（气泡）右侧；我发送的消息则将时间置于气泡左侧；时间不占用 `chat-bubble` 内部空间（放在气泡外侧）。

## 目录

1. 需求与验收标准
2. 实施方案
3. Todolist（时间戳原则）
4. 风险与回滚

## 1. 需求与验收标准

- 对方消息：时间显示在气泡右侧，垂直对齐气泡底部附近。
- 我发送的消息：时间显示在气泡左侧，垂直对齐气泡底部附近。
- 时间不放在 `chat-bubble` 内部，不影响气泡的文本布局与宽度控制。

## 2. 实施方案

- 调整消息行结构：新增 `chat-msg-main` 容器，内部为 `chat-bubble` + `chat-time-out`（气泡外时间）。
- 通过 `.chat-msg-row.me .chat-msg-main { flex-direction: row-reverse; }` 控制“我发送的时间在左侧”的效果。

## 4. 风险与回滚

- 风险：气泡最大宽度与时间同排时，极窄屏可能拥挤。
  - 处理：必要时可在移动端媒体查询下把时间改为单独一行或减小字体。
- 回滚：仅涉及样式与 DOM 结构调整，可通过 `git revert` 单提交回滚。
