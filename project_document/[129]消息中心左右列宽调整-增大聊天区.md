# [129]消息中心左右列宽调整-增大聊天区

## 1. 需求与验收标准
- 桌面端消息中心双栏布局中，左侧会话列表更窄，右侧聊天内容更宽。
- 不影响移动端单列布局与抽屉聊天。
- 不引入整页滚动（保持内部区域滚动与隐藏滚动条逻辑不变）。

## 2. 方案说明
- 通过调整 `.message-center-layout` 的 `grid-template-columns`：
  - 左列从 400px 缩小到 320px
  - 右列使用 `minmax(0, 1fr)` 确保可伸缩且避免内容溢出导致布局异常
