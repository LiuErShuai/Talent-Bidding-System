# 背景与目标
- 背景：当前登录/注册为独立页面，入口默认跳转 /login，用户需跳页；需求改为首页默认进入 home，头部按钮弹窗完成登录/注册，沿用现有表单逻辑且移除插画。
- 目标：1) 路由入口直达 home，/login 和 /register 不再展示页面；2) 头部提供登录/注册按钮，点击弹出对话框完成登录或注册；3) 复用现有校验与提交逻辑，移除插画；4) 保持未登录访问限制提示体验。
