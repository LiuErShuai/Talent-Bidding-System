# 背景与目标

- **背景**：当前私信输入框已固定高度，但仍偏高；同时提示语（placeholder）字号偏小，不够醒目。
- **目标**：进一步降低私信输入框高度，并适当增大提示语字号，保持“固定高度 + 内部可滚动编辑”的交互不变。

## 目录

1. 需求与验收标准
2. 实施方案
3. Todolist（时间戳原则）
4. 风险与回滚

## 1. 需求与验收标准

- 私信输入框高度再降低，仍保持固定高度。
- 内容超出仍可在输入框内部滚动编辑。
- 提示语（placeholder）字体更大一点，视觉更清晰。

## 2. 实施方案

- 将私信输入框 `rows` 由 3 调整为 2，并将输入框固定高度由 `80px` 调整为 `68px`。
- 通过 `::placeholder` 提升提示语字号（例如 15px）。
