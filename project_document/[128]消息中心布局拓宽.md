# [128]消息中心布局拓宽

> 背景与目标：用户反馈消息中心左右两栏“盒子太窄”，希望整体布局更宽一些，以提升会话列表与聊天内容的可读性与空间感。

## 1. 需求与验收标准
- 消息中心页头与主体最大宽度增加，整体两栏更宽。
- 左侧会话列表列宽适度加宽，标题与预览更易读。
- 不影响移动端单列布局与抽屉聊天展示。

## 2. 方案说明
- 将 `.center-header` 与 `.message-center-layout` 的 `max-width` 从 1100px 调整到 1280px。
- 将双栏列宽从 `360px 1fr` 调整为 `400px 1fr`，提升左侧列表可读性。
